About the platform

A modern banking frontend built for your Spring Boot APIs

What this UI delivers

This implementation focuses on the exact pages you requested: public marketing pages plus professional dashboards for customer, employee, and admin users.

Customer Dashboard

View saved IDs, load accounts, and use quick banking actions from a clean role-based workspace.

Employee Dashboard

Review operational counts and monitor customer and profile data from backend APIs.

Admin Dashboard

Create users, monitor roles, and review profiles and loan catalog counts.

Security Model

HTTP Basic Authentication with BCrypt-based backend validation and sessionStorage persistence.
